Short-form video generation and publishing tools for creators.

Video Poster helps creators organize media folders, build reusable text templates, generate vertical videos, schedule publishing, and connect posting workflows for their own TikTok accounts.

See How It Works

Features

Template-Based Videos

Create reusable templates with intro text, clip text, custom line styling, fonts, colors, and duration controls.

Folder Media Library

Scan creator-selected folders for shows, characters, intro clips, audio tracks, and reusable source footage.

Flexible Character Selection

Use all characters from each show or create template-specific selections across multiple shows and characters.

Scheduling Workflow

Configure a daily posting time and generate the next video from the creator's selected templates and media.

Workflow

  1. The creator selects local folders for audio, clips, and rendered output.
  2. The app scans show folders, character folders, and optional intro clips.
  3. The creator builds templates with text overlays, styling, and clip durations.
  4. The app renders vertical videos locally using the selected media and template rules.
  5. When connected and authorized, generated videos can be prepared for publishing through TikTok's posting workflow.

Responsible Posting

Creators are responsible for using media they own or are authorized to use. Video Poster is designed to publish only videos generated from creator-selected media and creator-configured settings.
